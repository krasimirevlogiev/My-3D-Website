BlindSight: Me and my friend Rumen made an app, which utilizes Monocular Depth Estimation AI model to assist blind people in pedestrial navigation. Our app offers object detection and guides the user to a safe path. On top of that we integrated voice commands so that the user can hear what is happening around him and since voice is not always fast enough we also integrated haptic feedback (available only with a devices with BLE connection). Our project won the Google Ideathon Bremen Chapter and we also participated in the Hult Prize and Google Solution Challenge 2024 and we are still waiting for results! Check out the app here

KnightShell: I made a unix type of shell written entirely in the most loved programming language in the past years: Rust. The shell has beautiful ASCII art acompanied with custom commands which the user can easily see using the help command. Hint: there is a little Easter Egg hidden somewhere and when found will display an interesting message. Check it out here
